Description

Cette méthode est une extension de Cell() permettant d'afficher du texte avec espacement de caractères ou compression horizontale.

CellFit(float w [, float h [, string txt [, mixed border [, int ln [, string align [, boolean fill [, mixed link [, boolean scale [, boolean force]]]]]]]]])

Les 8 premiers paramètres sont les mêmes que ceux de Cell(). Les paramètres supplémentaires sont :

scale

false : espacement de caractères
true : compression horizontale

force

false : espacer/compresser seulement si nécessaire (pas lorsque le texte est suffisamment court)
true : toujours espacer/compresser

Les quatre méthodes suivantes sont également fournies, permettant toutes les combinaisons de scale/force, et utilisant seulement les 8 paramètres de Cell() :

CellFitScale()
CellFitScaleForce()
CellFitSpace()
CellFitSpaceForce()
